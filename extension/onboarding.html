Krok 1 z 3
🔑 Token KSeF
Wklej token z portalu, resztę wykryjemy automatycznie
✏️
Token to 40-znakowy kod dostępu do API. Wyświetlany tylko raz – skopiuj go od razu.

  1. Zaloguj się do ksef.podatki.gov.pl (Profil Zaufany / e-Dowód / cert. kwalifikowany)
  2. Menu → Tokeny → Generuj token
  3. Nazwa tokenu: np. „KSeF Monitor"
  4. Uprawnienia: zaznacz wyłącznie „przeglądanie faktur"
  5. Kliknij Generuj → natychmiast kliknij Kopiuj
⚠️ Dla spółek wymagany druk ZAW-FA złożony wcześniej w US.
Tokeny ważne do 31.12.2026.
